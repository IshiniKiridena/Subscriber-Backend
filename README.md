# Subscriber-Backend

# Pre-requisites
- Install [Node.js](https://nodejs.org/en/)

# Getting started
- Clone the repository
```
git clone https://github.com/IshiniKiridena/Subscriber-Backend.git
```
- Install dependencies
```
cd <project_name>
npm install
```
- Run the project
```
npm run devStart
```
  Navigate to `http://localhost:3000`
